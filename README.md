# Student Productivity Analysis - Exploratory Data Analysis (EDA)

## Project Overview

This project aims to analyze factors affecting student productivity using Exploratory Data Analysis (EDA). The objective is to identify patterns and relationships between daily activities and productivity levels.

## Problem Statement

Students engage in various activities such as studying, sleeping, exercising, and using social media. This project investigates how these activities influence overall productivity.

## Dataset

The dataset contains information related to students' daily activities and productivity levels.

### Features Analyzed

* Study Hours
* Sleep Hours
* Exercise Duration
* Screen Time
* Productivity Level (High/Low)
* Other relevant variables

## Tools and Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## EDA Process

1. Data Cleaning
2. Handling Missing Values
3. Data Visualization
4. Correlation Analysis
5. Productivity Level Comparison
6. Insight Generation

## Key Insights

* Students with balanced study and sleep schedules tend to show higher productivity.
* Excessive screen time is associated with lower productivity.
* Physical activity appears to have a positive impact on productivity.

## Future Scope
